# Titus-Bot

Защита групп в телеграмме от спама

# Функции

1. Блокировка новых членов по спам базам
2. Удаление спам-сообщений по тексту
3. Белый список пользователей, прошедших проверку

# Описание

Бот защищает публичные группы от надоевшего спама тремя способами:

## 1. Сверка по списку известных спамеров

Используются публичные списки спамеров и каждый входящий пользователь проверяется по этому списку

Если новый пользователь оказывается в списке, он мгновенно блокируется и удаляется из группы

Спам-базы надежные и хорошо себя зарекомендовали, если пользователь есть в базе, то он 100% спамер

Единственный недостаток такого способа: бот может быть недавно зарегестрирован и пока что не попасть в такой список, тогда в дело вступает другой метод:

## 2. Определение спама по тексту сообщения

Бот анализирует, похоже ли сообщение на спам, ориентируясь на собранную базу спама

Если сообщение походит на спам, оно удаляется, но пользователь не блокируется,

Так, если это легитимный пользователь, ему будет предоставлен шанс повторно написать сообщение, которое точно так же анализируется по базе известного спама

На практике, боты пытаются несколько раз послать одно и то же сообщение, поэтому если они не прошли проверку первый раз, то не пройдут и второй

В то время, как легитимные пользователи перефразируют изначальное сообщение

Спам-база довольно надежная, верно определяет спам в 99,9% случаев, легитимные пользователи не замечают работы фильтра

Если пользователь прошел проверку по тексту, то он добавляется в глобальный список легитимных пользователей и больше не банится во всех группах, где подключен бот

## 3. Альфа-функция определения спама по изображениям

Данная функция находится в разработке, пока мало примеров такого спама

Бывает, что боты просто шлют в группу картинку без подписей (если подпись есть, она анализируется вторым методом), а на картинке уже размещены спам-текст

Тогда, бот через нейросеть распознает текст и что изображено на картинке и прогоняет это по собранной базе

Пока что примеров такого спама мало, поэтому бот пропускает такие изображения, но постепенно данные накапливаются и скоро эта функция выйдет из альфа-режима

# Как подключить бота в группу

Бота имеет смысл подключать только к публичным группам

1. Перейдите в настройки группы и
2. Нажмите «Добавить участников»
3. В поиске найдите `@titus_antispam_bot` и нажмите на элемент списка с этим ботом
4. Сделайте только что добавленного бота администратором, и
5. Добавьте ему права на «Удаление сообщений» и «Блокировку пользователей». Остальные права можно отключить, бот ими не будет пользоваться
6. Готово! Теперь, ваша группа под надежной защитой
